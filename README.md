# NGS-myhofe

An NGSolve add-on project for teaching how to implement high-order finite elements


Installation:

    git clone https://github.com/TUWien-ASC/NGS-myfe.git
    cd NGS-myfe
    mkdir build
    cd build
    cmake ..
    make -j4 install


Quick installation:

    pip3 install git+https://github.com/TUWien-ASC/NGS-myhofe -v
